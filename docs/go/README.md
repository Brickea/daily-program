# Go 学习体系

## 学习路径
### 入门
- 语法基础、模块与包管理、常用标准库。
### 进阶
- 并发模型（goroutine/chan）、测试与基准、常见工具链。
### 高级
- 性能优化、分布式与微服务实践、调试与运维。
